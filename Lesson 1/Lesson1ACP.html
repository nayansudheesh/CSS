What i know about CSS

CSS stands for cascading style sheet

There are 3 types of CSS

  1. Inline CSS , In this method , the CSS is written inside the element using the 'style=' tag. example: < h1 style="text-align: center;> < /h1>
  2. External CSS: in this method the CSS is writen in an external file which is linked to the main page using < link rel="stylesheet" href="insert.css">
  3. Internal CSS: In this method , code is written inside the < style> tag.
Uses of CSS

CSS is used to style websites,that is, to give color to text , change font family , change background color etc.
